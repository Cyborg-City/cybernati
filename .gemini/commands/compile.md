# /compile

Executes the atomic writing and cross-linking phase of the Ingestion Loop.

## Instructions
1. **Verify Approval**: Confirm a 'Green Light' was given in the current session log.
2. **Sequential Mapping**: Check `private/vault-schema.md` and current directories for ID sequences.
3. **Execution (Serious Write)**:
   - Create/Update notes using `templates/TMP-0000-vault-property-schema.md`.
   - Apply naming convention: `PREFIX-####-kebab-case.md`.
   - Cross-link all atomic notes back to the Evidence summary.
4. **The Redaction Rule**: 
   - Use `||lorem ipsum||` for aesthetic redaction only. 
   - **Never** obscure real names, dates, or data points.
5. **Aesthetic Refinement (Optional)**:
   - Ask the User: "Shall I perform a Cybernati™ Flavor Pass on these notes?"
   - If approved, apply memo headers, corporate terminology, and aesthetic redactions.
6. **Final Log**: Update `agent-notes/agent-log.md` with created IDs and action summary.
7. **Report**: Provide a final summary of files to the User.
